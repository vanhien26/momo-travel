# momo-travel — Claude Context

## Stack
- Next.js (xem package.json để biết version)
- Tailwind CSS
- TypeScript

## Lệnh thường dùng
```bash
npm install    # Cài dependencies
npm run dev    # Chạy dev server
npm run build  # Build production
```

## Cấu trúc thư mục
- src/app/        — App Router pages
- src/components/ — Reusable components
- public/         — Static assets

## Nguyên tắc khi code
- Giữ nguyên cấu trúc URL hiện có, không tự đổi route
- Comment bằng tiếng Việt nếu logic phức tạp
- Không xóa code cũ, comment out nếu cần thay thế
